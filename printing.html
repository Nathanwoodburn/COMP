My 3D Printers

Click on a printer to find out more about it

FDM Printer

This printer is a Fused deposition modeling (FDM) printer. It creates the object by squirting molten plastic in the shape of the object.
It is the most common and cheapest type of 3d printer.



Resin Printer

This printer is a Masked Stereolithography Apparatus (MSLA) printer. It uses a light sensitive resin. The liquid resin hardens when exposed to light.
The printer shines an image on the resin with the use of a LCD screen and a UV light array. The resin then hardens in the shape of the image.



Resin Cleaner

Resin prints are not fully hardend when they are removed from the printer. This machine cleans the drips of resin from the print to remove artifacts.
It does this using isopropyl alcohol. It then shines a bright UV light on the part hardening it fully.
